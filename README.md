# DisplayPilot
An open-source dark/light theme and monitor brightness tool for Windows 10/11 desktops.

The current development build includes a read-only WinUI 3 display inventory with
DDC/CI brightness probing for external monitors and WMI brightness probing for
internal panels. See [Display-path verification](docs/display-path-verification.md)
for build, run, and test instructions.
